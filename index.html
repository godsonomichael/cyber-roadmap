Splunk SPL Practice Queries
🔍 These are real Splunk Search Processing Language (SPL) queries Nigerian SOC employers test in interviews. Practice these in the free Splunk Trial or TryHackMe's Splunk room. Knowing SPL = instant credibility.
🔐 Authentication & Failed Login Detection
Find all failed login attempts
index=main sourcetype=WinEventLog EventCode=4625
| stats count by src_ip, user
| where count > 5
| sort -count
Brute force detection — 10+ failures in 5 minutes
index=security sourcetype=WinEventLog EventCode=4625
| bin _time span=5m
| stats count by _time, src_ip, user
| where count >= 10
| rename count as "Failed Attempts"
Successful logins after multiple failures (credential stuffing)
index=security (EventCode=4625 OR EventCode=4624)
| stats count(eval(EventCode=4625)) as failures,
count(eval(EventCode=4624)) as successes by src_ip, user
| where failures > 5 AND successes > 0
| eval risk="High"
🌐 Network & Firewall Analysis
Top talkers — highest network traffic by source IP
index=network sourcetype=firewall
| stats sum(bytes) as TotalBytes by src_ip
| eval TotalMB=round(TotalBytes/1048576,2)
| sort -TotalMB
| head 20
Blocked/denied firewall connections
index=firewall action=blocked
| stats count by src_ip, dest_ip, dest_port
| where count > 10
| sort -count
Port scan detection — single IP hitting many ports
index=network
| stats dc(dest_port) as unique_ports by src_ip
| where unique_ports > 20
| eval threat="Possible Port Scan"
| sort -unique_ports
🛡️ Threat Detection & Alert Triage
Malware hash lookup with VirusTotal enrichment placeholder
index=endpoint sourcetype=sysmon EventCode=1
| stats count by MD5, Image, CommandLine
| lookup malware_hashes MD5 OUTPUT threat_name
| where isnotnull(threat_name)
Suspicious PowerShell execution (common attacker tool)
index=endpoint sourcetype=WinEventLog EventCode=4688
| where match(CommandLine, "(?i)(encodedcommand|bypass|hidden|invoke-expression|iex|downloadstring)")
| table _time, Computer, user, CommandLine
| sort -_time
Critical security alerts in last 24 hours
index=alerts severity=critical earliest=-24h
| stats count by alert_name, src_ip, dest_ip
| sort -count
| head 25
📊 SOC Dashboard Queries
Real-time event volume by severity (for SOC dashboard)
index=* earliest=-1h
| eval severity=if(isnull(severity),"info",severity)
| timechart span=5m count by severity
MITRE ATT&CK technique mapping
index=endpoint
| stats count by mitre_technique, mitre_tactic
| sort -count
| eval risk_score=case(count>100,"Critical", count>50,"High", count>10,"Medium", true(),"Low")
User account created outside business hours (insider threat)
index=security EventCode=4720
| eval hour=strftime(_time,"%H")
| where hour < 8 OR hour > 18
| table _time, user, Account_Name, Computer
